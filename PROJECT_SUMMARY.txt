/*
 * CHAT UI CLONE - PROJECT SUMMARY
 * ================================
 * 
 * This document provides an overview of the complete Chat UI Clone app.
 * 
 * PROJECT OVERVIEW:
 * -----------------
 * A fully functional Flutter chat application with local message management,
 * AI bot auto-replies, and modern UI design inspired by Messenger/WhatsApp.
 * 
 * FILE STRUCTURE:
 * ---------------
 * 
 * 1. lib/main.dart
 *    - Entry point of the application
 *    - Initializes MaterialApp with custom theme
 *    - Routes to ChatScreen
 * 
 * 2. lib/models/message_model.dart
 *    - Message data model with properties:
 *      • sender (String) - Name of message sender
 *      • text (String) - Message content
 *      • timestamp (DateTime) - When message was sent
 *      • isUser (bool) - Differentiates user from bot
 *    - Includes JSON serialization methods for future use
 * 
 * 3. lib/theme/app_theme.dart
 *    - Centralized theme configuration
 *    - Color palette (Messenger blue theme)
 *    - Text styles using Google Fonts (Poppins)
 *    - Light and dark theme support
 * 
 * 4. lib/widgets/message_bubble.dart
 *    - Individual message display component
 *    - Features:
 *      • Dynamic alignment (left for bot, right for user)
 *      • Different bubble colors based on sender
 *      • Rounded corners with proper border radius
 *      • Avatar icons for both user and bot
 *      • Timestamp display below each message
 *      • Soft shadows for depth effect
 * 
 * 5. lib/widgets/message_input.dart
 *    - Message input component
 *    - Features:
 *      • TextField with hint text
 *      • Send button with state management (enabled/disabled)
 *      • Emoji button (placeholder for future feature)
 *      • Attachment button (placeholder)
 *      • Keyboard unfocus on send
 *      • Submit on Enter key
 * 
 * 6. lib/screens/chat_screen.dart
 *    - Main chat interface (StatefulWidget)
 *    - Features:
 *      • AppBar with bot profile, avatar, and online indicator
 *      • Scrollable message list using ListView.builder
 *      • Auto-scroll to latest message
 *      • Typing indicator animation
 *      • Auto-reply system with 1-second delay
 *      • Contextual bot responses based on keywords
 *      • Initial welcome message
 *      • SafeArea for proper layout
 * 
 * KEY FEATURES:
 * -------------
 * 
 * ✅ Local Message Storage
 *    - Messages stored in List<Message>
 *    - No backend required
 *    - Real-time updates with setState()
 * 
 * ✅ Smart Auto-Reply
 *    - Detects greetings, questions, thanks
 *    - Context-aware responses
 *    - Random fallback messages
 *    - 1-second typing delay for realism
 * 
 * ✅ Modern UI Design
 *    - Messenger-style blue theme
 *    - Rounded message bubbles
 *    - Soft shadows and gradients
 *    - Professional avatar and icons
 *    - Online status indicator
 * 
 * ✅ Smooth UX
 *    - Auto-scroll to latest message
 *    - Animated typing indicator
 *    - Keyboard management
 *    - Responsive button states
 *    - No keyboard overflow
 * 
 * ✅ Clean Code
 *    - Well-organized file structure
 *    - Separation of concerns
 *    - Reusable components
 *    - Comprehensive comments
 *    - Following Flutter best practices
 * 
 * DEPENDENCIES:
 * -------------
 * • google_fonts: ^6.1.0 - Poppins font family
 * • intl: ^0.19.0 - Date/time formatting
 * • cupertino_icons: ^1.0.8 - iOS-style icons
 * 
 * HOW TO RUN:
 * -----------
 * 1. Ensure Flutter SDK is installed (3.9.2+)
 * 2. Open terminal in project directory
 * 3. Run: flutter pub get
 * 4. Run: flutter run
 * 5. Select your target device
 * 
 * CUSTOMIZATION:
 * --------------
 * • Colors: Edit lib/theme/app_theme.dart
 * • Fonts: Change GoogleFonts in app_theme.dart
 * • Bot responses: Modify _getBotResponse() in chat_screen.dart
 * • UI layout: Adjust widgets in chat_screen.dart
 * 
 * TESTING:
 * --------
 * • Basic widget test included in test/widget_test.dart
 * • Run tests with: flutter test
 * 
 * SUPPORTED PLATFORMS:
